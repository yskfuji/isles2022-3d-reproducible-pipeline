# ISLES 2022 public snapshot — `isles2022-v1.0-interview`

## Purpose
A stable portfolio snapshot for hiring review and external technical inspection.

## Snapshot type
Stable review build.

## Reproduced scope
- ISLES 2022 lesion segmentation preprocessing, training, and evaluation
- 3D U-Net based pipeline with threshold and postprocess sweeps
- Size-stratified lesion analysis and reproducibility-oriented reporting

## Main strengths in this snapshot
- Audit-ready documentation
- Explicit quickstart for preprocess → train → evaluate
- Threshold and connected-component sweep support
- Portfolio reports for local validation and test-style evaluation

## Known limitations
- Protected medical data is not bundled
- GitHub Release text still needs to be published in the web UI
- Smoke test validates repository wiring, not full model quality
