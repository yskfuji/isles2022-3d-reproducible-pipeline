# ISLES 2022 公開スナップショット  `isles2022-v1.0-interview`

## 目的
採用選考や外部レビューで安定して確認してもらうための公開スナップショットです。

## スナップショットの位置づけ
安定版のレビュー用ビルドです。

## レビュー開始時の導線
- README: リポジトリ全体の概要とクイックスタート
- docs/reproducibility_checklist.md: 外部レビュー向けの確認項目と合格条件
- AUDIT_MAP.md: 主要な公開成果物や実行コマンドの所在

## 再現対象の範囲
- ISLES 2022 病変セグメンテーションの前処理、学習、評価
- 3D U-Net ベースのパイプラインと、しきい値・後処理のスイープ
- 病変サイズ別の解析と、再現性を意識したレポート

## このスナップショットの主な強み
- 監査しやすいドキュメント構成
- 前処理 → 学習 → 評価までを明示したクイックスタート
- しきい値と連結成分のスイープに対応
- ローカル検証と test 風評価の両方に向けたポートフォリオ資料

## 既知の制約
- 保護対象の医療データは同梱していません
- GitHub Release の本文はこの原稿ファイルと同期しています
- smoke test はリポジトリ配線の確認用であり、モデル品質そのものを保証するものではありません
